What is ATHENA?
..ATHENA is an APP Packaging Architecture built for Ubuntu Linux by Apollon Data Metrics Agency. It is used to run Linux Shell Apps and HTML5 Web Apps on Linux Systems as well as Some Alternative Systems such as MacOS and Windows. All ADM Apps require that the users have ATHENA installed to run its shell apps. Getting started is simple, and can be done by following our Installation Guide.. After Installation, you will ba able to download and run any of our shell apps found in the designated Platform Sections of this site..
